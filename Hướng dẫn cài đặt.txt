Lưu ý: Trước khi cài đặt thì đảm bảo rằng bạn đã cài đặt nodejs và npm, câu lệnh cài đặt expo CLI:
"npm install --global expo-cli" hoặc "yarn global add expo-cli"
## Trường hợp chạy với localhost 
1) Mở thư mục hiện tại trên Visual Studio Code.
2) Mở terminal và di chuyển đến thư mục "client" bằng lệnh "cd client", sau đó cài đặt các gói phụ thuộc bằng lệnh "npm install" hoặc "yarn".
3) Quay lại thư mục mẹ bằng lệnh "cd ..", sau đó di chuyển đến thư mục "server" và cài đặt các gói phụ thuộc tương tự.
4) Sau khi cài đặt xong, mở hai terminal và chia màn hình thành hai phần.
5) Trong terminal đầu tiên, chạy lệnh "npm run watch" hoặc "yarn watch" để theo dõi các thay đổi trong mã nguồn.
6) Trong terminal thứ hai, chạy lệnh "npm run server" hoặc "yarn server" để khởi động máy chủ.
7) Đảm bảo rằng máy chủ đang chạy.
8) Mở thêm một termianl mới và quay lại thư mục "client", chạy lệnh "npx expo start" để khởi động dự án React Native bằng Expo.
9) Vào file .env tại thư mục client, thay đổi biến môi trường có tên "EXPO_PUBLIC_AUTH_API" thành địa chỉ IPV4 mà đang kết nối và thêm đuôi :3000.
Ví dụ như sau nếu địa chị IPV4 của bạn là 192.168.1.12 thì sửa thành http://192.168.1.12:3000
10) Quét mã QR trên terminal client sau khi ứng dụng đã được build để sử dụng ứng dụng 
Note: Nếu tại bước này bạn bị yêu cầu phải đăng nhập vì đây là dự án expo và được bảo mật chỉ người phát triển có thể chạy ứng dụng
